## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- NPM

### Setup

- git https://github.com/Isabellanamutenda/hello-react-front-end.git
- cd hello-react-front-end

### Install

- yarn

### Usage

- npm run start
